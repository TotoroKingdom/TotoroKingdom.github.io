linux基本命令（命令可以组合使用）

sync # 将数据从内存同步到硬盘

shutdown #关机

reboot #重启

halt # 关闭系统

ls # 查看当前目录

ls -a #查看所有目录，包括隐藏目录

ls -l #查看所有文件，包含文件的属性和权限，不包括隐藏文件

cd # 切换目录

cd.. # 返回上一级目录

cd~ #回到当前用户目录

pwd #显示当前用户所在的目录

mkdir #创建目录

mkdir -p #创建多级目录

rmdir #删除目录

rmdir -p #删除多级目录

rm #删除文件

rm -f #忽略不存在的文件，强制删除

rm -r #递归删除目录

rm -i #互动，询问是否删除

rm -rf / #删除系统中所有的文件

cp #复制文件或者目录

mv #移动文件或者目录

mv -f #强制

mv -u #只更换已经更新过的文件

chgrp #更改文件属组

chown #更改文件属主，也可同时更改文件属组

==chmod #更改文件9个属性==

r=4 w=2 x=1

ln #创建硬链接

ln -s #创建软链接

touch #创建文件

echo #输入字符串

查看文件中的内容的命令

cat 由第一行开始显示文件内容

tac 从最后一行开始显示

nl 显示的时候，顺道输出行号

more 一页一页的显示文件内容

less 与more类似，可以往前翻页

head 只看头几行

tail 只看尾部几行

linux用户管理

useradd 添加用户

useradd -m 自动添加主目录

userdel 删除用户

userdel -r 删除用户的时候将目录一并删掉

linux用户组管理

groupadd 添加用户组

groupadd -g 添加用户组并指定端口号

groupdel 删除用户组

groupmod 修改用户组

linux磁盘管理

df 列出文件系统整体的磁盘使用量

du 检查磁盘空间使用量

mount 挂载

umount 卸载

linux进程管理

ps 查看当前系统中正在执行的各种进程的信息

ps -a 显示当前终端运行的所有的进程信息

ps -u 以用户的信息显示进程

ps -x 显示后台运行进程的参数

grep 查找文件中符合条件的字符串

| 管道符

ps -aux|grep mysql 查看所有与mysql相关的进程

ps -ef 查看父进程的信息

pstree -pu 显示父id和显示用户组

export PATH CLASSPTH JAVA_HOME
