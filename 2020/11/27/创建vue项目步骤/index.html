创建vue项目步骤(需使用管理员模式)

1—新建.vue文件

vue init webpack hello-vue

2—进入工程目录

cd hello-vue

3—安装vue-router

npm install vue-router —save-dev

4—安装element-UI

npm i element-ui -S

5—安装依赖

npm install

6—安装SASS加载器

npm install sass-loader node-sass —save-dev

7—启动测试

npm run dev

8—在idea中打开项目
